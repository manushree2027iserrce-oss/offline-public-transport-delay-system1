# Offline Public Transport Delay Attribution System

## Project Overview
This system detects and analyzes delays in public transport such as buses and trains even without internet connectivity.

## Features
- Offline data collection
- Delay prediction using machine learning
- Cause identification (traffic, weather, breakdown)
- Simple dashboard for monitoring delays

## Technologies Used
- Python
- Machine Learning
- HTML / CSS
- GitHub

## Use Case
Helps transport authorities and passengers understand why delays happen and improve scheduling.
